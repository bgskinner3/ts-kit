# TypeLens

A modular collection of reusable, **type-safe TypeScript utility functions** for arrays, objects, validation, formatting, and more.

Built with TypeScript and tested with Jest, this library emphasizes **type safety**, **clean code**, and **reusability**.

---

## Features

- Type-safe validations, parsers, and assertions
- Array, object, and string helper functions
- Core and domain-specific utilities
- Fully tested with Jest
- Linting and formatting with ESLint + Prettier

---

## Installation

```bash
# If published on npm
npm install my-ts-utils

# Or clone directly
git clone https://github.com/<your-username>/my-ts-utils.git
```
